# BLE 设备控制小程序

## 简介

这是一个基于微信小程序的蓝牙设备控制应用，主要用于连接和控制智能 BLE 设备。

## 主要功能

- **设备扫描与连接**：通过输入设备名称，扫描并连接附近的 BLE 设备
- **设备类型支持**：支持两种设备模式
  - LED灯：简单的开关控制功能
  - RGB灯：支持开关、颜色设置和多种工作模式
- **RGB灯控制功能**
  - 开/关总开关
  - 自定义颜色（RGB值）
  - 多种预设模式（关闭、呼吸灯、阅读暖光、影院蓝光、晚安渐变）
- **实时调试信息**：显示设备连接和控制过程中的详细日志

## 使用说明

1. 在小程序中输入要连接的设备名称
2. 根据设备类型选择相应的控制功能
3. 点击"开始扫描"按钮搜索设备
4. RGB灯可设置颜色和选择工作模式

## 注意事项

- 需要设备开启蓝牙权限才能使用扫描功能
- 确保目标 BLE 设备处于可发现状态
